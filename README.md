# 📊 Social Media Engagement Analysis Dashboard

## 🚀 Project Overview

This project analyzes social media engagement data using Power BI, Python, and SQL to uncover key insights such as content performance, engagement trends, and optimal posting times.

---

## 📌 Key Insights

* 📈 Total Engagement: 31K+
* 📊 Avg Engagement Rate: 12.67%
* 🎥 Reels generate the highest engagement
* ⏰ Peak engagement observed during mid-day & evening hours
* 📝 Medium-length captions perform better than short/long captions

---

## 🛠 Tools Used

* Power BI (Dashboard Visualization)
* Python (Pandas, Matplotlib)
* Excel / CSV (Data Source)

---

## 📂 Project Structure

* `data/` → Dataset (social_data.csv)
* `python/` → Data analysis notebook
* `powerbi/` → Power BI dashboard file
